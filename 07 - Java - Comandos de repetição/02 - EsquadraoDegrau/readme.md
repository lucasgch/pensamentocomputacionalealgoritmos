# EsquadraoDegrau.java 

Um inteiro positivo é membro do Esquadrão do Degrau se puder ser expresso pela soma de uma sequência de números naturais consecutivos, começando por 1. Por exemplo, 10 é membro do Esquadrão do Degrau, pois 10 = 1 + 2 + 3 + 4.

Elabore um programa que `LÊ DO TECLADO` um inteiro positivo `n` e `ESCREVE NA TELA` se `n` é membro do Esquadrão do Degrau ou não. Para tanto, IMPLEMENTE O SEGUINTE MÉTODO:

`boolean ehDegrau(int n):`
- Retorna true, se `n` é membro do Esquadrão do Degrau, ou false, caso contrário.

## Restrições:

- Apenas o método `main()` deve interagir com o usuário (ou seja, ler e escrever na tela). Utilize o comando while.

## Exemplos

Entre com um inteiro positivo: 15
É membro do Esquadrão do Degrau

Entre com um inteiro positivo: 12
Não é membro do Esquadrão do Degrau

Entre com um inteiro positivo: 1
É membro do Esquadrão do Degrau

Entre com um inteiro positivo: 666
É membro do Esquadrão do Degrau
