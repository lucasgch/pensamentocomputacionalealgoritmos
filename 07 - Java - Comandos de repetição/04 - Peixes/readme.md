# Exercício em sala: Peixes.java

Cada espécie de peixe possui uma faixa de pH adequada para viver. Por exemplo, o peixe palhaço necessita de águas com pH entre 7.8 e 8.4; já o betta prefere águas com pH entre 6.0 e 8.0.

Elabore um programa que LÊ DO TECLADO a faixa de pH adequada para uma espécie de peixe, um inteiro positivo n indicando o número de medições de pH e, em seguida, n valores de pH. O programa deve ESCREVER NA TELA quantas medições estão com o pH fora da faixa adequada.

## Restrição

- Utilize o comando for .

## Exemplo de execução

```text
Entre com o pH mínimo: 7.8
Entre com o pH máximo: 8.4
Entre com o número de medições: 5
Entre com a medição 1: 7.9
Entre com a medição 2: 8.5
Entre com a medição 3: 8.3
Entre com a medição 4: 7.7
Entre com a medição 5: 8.4
Número de medições fora da faixa: 2
```
