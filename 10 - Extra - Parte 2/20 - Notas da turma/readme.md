# Notas da turma

Em uma determinada disciplina, a média final do aluno é dada pela média aritmética simples das notas das provas realizadas, sendo o aluno aprovado se sua média final for maior ou igual a 6,0.

## Tarefa

Escreva um programa que lê do teclado o número de alunos da turma, o número de provas aplicadas pelo professor, bem como as notas das provas de cada aluno e escreve na tela a média final de cada aluno, indicando se o aluno foi aprovado ou reprovado e, ao final, a média da turma (média das médias).

## Restrições

- Utilize comandos for aninhados.
- Não utilize arranjos.

## Formato

Entre com o número de alunos: {int}
Entre com o número de provas: {int}

Entre com as notas do aluno 1: {float} ⋯ {float}
Aluno 1 [aprovado|reprovado] com média {float}
⋮
Entre com as notas do aluno n: {float} ⋯ {float}
Aluno n [aprovado|reprovado] com média {float}

A média da turma foi {float}
Apresente as médias com uma casa decimal.
