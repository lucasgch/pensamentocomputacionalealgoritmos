# Calculadora.java

- (Enunciado)[https://nobrega.prof/disciplinas/alg/slides/java-1/#/68]

Elabore um programa que LÊ DO TECLADO uma expressão aritmética composta por um número real, um caractere representando uma operação matemática e outro número real (nesta ordem e separados por espaço), e ESCREVE NA TELA o resultado da operação.

- Operações possíveis: 
  - adição ( + )
  - subtração ( - )
  - multiplicação ( * )
  - divisão ( / ).
  
Em caso de operação inválida ou divisão por zero, o programa deve reportar um ERRO.

## Restrição:

Utilize switch como expressão.

## Exemplos

```
Entre com a expressão: 2 * 3.14
Resultado: 6.28

Entre com a expressão: 30 + 12
Resultado: 42.0

Entre com a expressão: 1 / 0
Erro: Divisão por zero

Entre com a expressão: 3 # 4
Erro: Operação inválida
```
