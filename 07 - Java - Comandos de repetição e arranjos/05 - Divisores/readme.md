# Exercício em sala: Divisores.java

Elabore um programa que LÊ DO TECLADO um inteiro positivo n e ESCREVE NA TELA todos os divisores de n, bem como se n é um número primo ou não.

## Restrições:

Utilize o comando for .
A lista de divisores deve ser escrita entre colchetes, com os divisores separados por vírgula.

## Exemplos de execução:

```text
Entre com um inteiro positivo: 8128
Divisores: [1, 2, 4, 8, 16, 32, 64, 127, 254, 508, 1016, 2032, 4064, 8128]
Não é primo

Entre com um inteiro positivo: 82589933
Divisores: [1, 82589933]
É primo

Entre com um inteiro positivo: 1
Divisores: [1]
Não é primo

Entre com um inteiro positivo: 57
Divisores: [1, 3, 19, 57]
É primo de Grothendieck
```
