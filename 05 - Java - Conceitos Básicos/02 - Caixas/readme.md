# Exercício em sala: Caixas.java

Considere uma fábrica que produz caixas de papelão em formato de paralelepípedo. Elabore um programa que **LÊ DO TECLADO** as dimensões de uma caixa (em m) e **ESCREVE NA TELA** a área da superfície (em m2) e o volume (em m3) da caixa.

## Exemplo de execução:

```
Entre com a altura (m): 1.0
Entre com a largura (m): 4.0
Entre com a profundidade (m): 9.0
A área é 98.0 m2 e o volume é 36.0 m3.
```
