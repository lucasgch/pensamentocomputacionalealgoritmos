# Exercício em sala: `Altura.java`

## `Altura1.java`

Elabore um programa que **LÊ DO TECLADO** a altura de uma pessoa em pés + polegadas e **ESCREVE NA TELA** a altura correspondente em metros, com duas casas decimais. Lembre-se de que

1 POL = 2.54 cm
1 PÉ = 12pol


**Exemplos de execução:** `1`  `2`

> Entre com sua altura em pés + polegadas (ex: 5 10): 5 8
>
> Altura em metros: 1.73
